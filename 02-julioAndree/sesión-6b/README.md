# 16/04/26 Arduino y Physical Computer  
## Interacción  
def: Proceso iterativo de pensar, escuchar y hablar. 
  
Input / Proceso = Output  
  
**D/I A/I** Entradas  
**P** Proceso  
**D/O A/O** Salidas  
Tanto entradas como salidas pueden ser digitales o analógicas.  
### Arduino  
Arduino es una placa basada en el chip ATmega168  
* 13 pines digitales (6 Output/ Salida PWM)
   * PWM: Pulse With Modulations (Puede ser analógo) Nos da un promedio entre 0 y 1/altos y bajos por pulso eléctricos. (Osea que nos da 0,1 0,2, etc...)
* Simbolo ~: Es analógo, sirve para entradas o salidas través del código. **Analógo: Escala de grises** **Digital: Blanco y Negro**
