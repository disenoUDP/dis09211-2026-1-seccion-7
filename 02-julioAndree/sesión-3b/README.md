# 27/03/2026  
El jueves llegue a clases y el drawdio dejó de funcionar!!  
Pensé q pudo ser la bateria así que fuí a cargarlo, la clase de hoy fue sobre el Hacking, vimos referentes y materiales.

Luego de observar bien el prototipo del bolso decidimos hacer unos cambios para la entrega final, sabiendo que tenemos que incorporar el circuito a esté decidimos que el bolso tuviera una cambio de imagen.  
### Cambios:  
   * Cambio de material: Cuerina para la parte del bolso y tela polar para la parte del asa. La tela polar simula la parte de peluche del juguete abrazador mono y la cuerina roja es una referencia a la chinita. En si la elección de colores esta inspirado en el insecto.
   * Cambio en estructura: Descartamos el velcro para la parte del asa devido a que este se puede soltar al cargar mucho peso, simplificamos el concepto de los brazos del mono y decidimos hacer una sola asa rellena para que sea cómoda y "abrazadora". Dentro del bolso habrá un compartimiento para el círcuito protoboard y un pequeño orificio para el botón de la alarma.
